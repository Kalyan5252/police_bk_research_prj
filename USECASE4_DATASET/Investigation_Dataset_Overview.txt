📂 Dataset Directory Structure Provided to Participants

The following folders contain structured and semi-structured datasets for use in solving the investigation use case. These reflect real-world data types collected during narcotics and financial crime investigations:

1. 📁 Bank Statements
   - Contains Excel/CSV files of banking transactions.
   - Used to analyze post-transport financial disbursements.

2. 📁 CDRs (Call Detail Records)
   - Logs of telecom activity including A-Party, B-Party, call duration, and cell location.
   - Used to track suspect movements and call relationships.

3. 📁 IPDRs (Internet Protocol Detail Records)
   - Logs of data sessions, including app usage and IP addresses.
   - Used to detect usage of encrypted apps like Threema, Telegram.

4. 📁 SDR (Subscriber Detail Records)
   - Contains lists of phone numbers with registration addresses and SIM activation details.

5. 📁 SMS_Headers
   - Logs of incoming SMS headers for financial alerts and app-based communication.

6. 📁 TrueCallers
   - Truecaller-based identity resolution of mobile numbers.
   - Used for alias tracking and identity validation.

7. 📁 CAFs (Customer Application Forms)
   - Documents showing SIM registration identity and address 
   - Used to detect fraudulent SIM registrations or dummy identities.

8. 📁 KYC Details
   - Bank KYC documents for identified account holders.
   - Used for profiling financial suspects and Hawala operators.\
